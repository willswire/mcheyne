# The M'Cheyne Plan

Native iOS app for following the M'Cheyne Bible reading plan.

## TestFlight

TestFlight is Apple's official beta distribution platform for iOS apps.
Public beta: [https://testflight.apple.com/join/3yeqwSrj](https://testflight.apple.com/join/3yeqwSrj)
How to join:
- Install the TestFlight app from the App Store.
- Open the public beta link and accept the invite.
- Install the beta build from TestFlight.

## Requirements

- Xcode
- iOS 17+

## Development

Open `The M'Cheyne Plan.xcodeproj` in Xcode and run.

## Structure

- `App/` - App source and shared assets
- `Tests/` - Unit tests
